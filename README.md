

# Netflix User Behavior Analysis

## Project Overview
Analyzed Netflix content data to identify viewer trends and content patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Movies dominate Netflix content.
- TV-MA is the most common rating.
- Drama is the most popular genre.
- USA contributes the highest amount of content.
- Netflix content increased significantly after 2015.

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Visualization
5. Business Insights
